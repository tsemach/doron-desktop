# Project Naming

- The product's name is **Amicus**. "Doron Desktop" is a temporary/legacy name from before the rename.
- When referring to the project in conversation, docs, comments, UI copy, or any other non-resource context, use **Amicus**.
- Exception: real, physical resources that were provisioned under the old name — the GitHub repo, the local directory path (`doron-desktop`), domains, package names, app identifiers (e.g. `com.tsemach.doron-desktop`), database names, etc. — must keep using `doron-desktop` until they are manually migrated. Do not rename these on your own initiative.
